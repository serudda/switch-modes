## The Problem

- AI is expensive. GPT-4/Claude Opus costs add up fast
- Most tasks don't need premium models
- But switching models manually is tedious
- Users overpay because it's easier than optimizing

**The pain:** You're paying $0.60/1M tokens when $0.08/1M would work fine.

---

## The Solution

**Model Modes** - Dynamic model switching for OpenClaw

4 modes. Simple commands. Instant switching.

```
eco mode       ‚Üí Cheapest (summaries, quick Q&A)
balanced mode  ‚Üí Daily driver (general work)
smart mode     ‚Üí Powerful (complex reasoning)
max mode       ‚Üí Maximum (critical tasks)
```

**Real savings:** 60-80% cost reduction reported by users.
